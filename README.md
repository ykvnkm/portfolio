# Портфолио UX/UI дизайнера

Это мое портфолио, демонстрирующее мои проекты в области UX/UI и графического дизайна.

## Проекты

- **AIST Filter** - Веб-приложение для фильтрации конфиденциальных данных
- **CUPCAST** - Интерфейс предсказательной модели для кофейных автоматов
- **KOLPAK NEWS** - Новостной агрегатор STEM-направления
- **Rentbuddy** - Сервис поиска соседей для студентов

## Технологии

- HTML5
- CSS3
- JavaScript
- Адаптивный дизайн

## Локальный запуск

1. Клонируйте репозиторий:
```bash
git clone https://github.com/yourusername/portfolio.git
```

2. Установите зависимости:
```bash
npm install
```

3. Запустите локальный сервер:
```bash
npm start
```

4. Откройте [http://localhost:3000](http://localhost:3000) в браузере

## Деплой

Для деплоя на GitHub Pages используйте:
```bash
npm run deploy
```
